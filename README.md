# E-goshopping
---

E-go Shopping is a online shopping website. 
It uses a distributed system architecture . The core technical stacks are shown as below
- Core technical stacks
    - Spring Security
    - Spring Boot  
    - Spring Cache
    - Redis
    - Spring Session
    - MybatisGenerator
    - Mysql 
    - Dubbo
    - Zookeeper
    - FastDFS
    - Nginx
    - Kinkeditor
    - RabbitMq
    - PageHelper
    - Solr
    - Mycat
